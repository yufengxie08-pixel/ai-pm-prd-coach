# Example Output

## Sample task

输入想法：
做一个给大学生用的 AI 面试模拟助手，能根据岗位生成模拟问题并给反馈。

## Expected interaction style

The skill should first ask clarification questions such as:

1. 目标用户是所有大学生，还是以求职中的应届生为主？
2. 你希望先支持哪些岗位类型，比如产品、运营、技术还是通用岗位？
3. 反馈重点更偏回答内容质量、表达能力，还是岗位匹配度？
4. 这个功能的主要使用场景是面试前练习，还是平时学习提升？
5. 你希望 MVP 先做文字问答，还是包含语音模拟？

## Expected PRD shape

The final result should include:

- 明确的目标用户与求职场景
- 收敛后的 MVP 范围
- 2 到 4 个核心功能模块
- 至少 3 条用户故事
- 至少 5 条可测试验收标准
- 至少 3 个 AI 相关风险或假设
